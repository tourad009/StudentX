# StudentX

Ce projet est une application web pour la gestion des étudiants, de leurs évaluations et de leurs notes.

## Technologies utilisées

- **PHP** (Backend)
- **MySQL** (Base de données)
- **Bootstrap** (Frontend)
- **HTML/CSS/JS** (Frontend)
